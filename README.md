# SharpShooter

A first-person shooter prototype developed in Unity 6 using C#.

## Overview

SharpShooter is an FPS project created as the final project of the Complete C# Unity 3D Game Development in Unity 6 course.

The game features multiple weapon types, AI enemies, pickups, projectile combat, shield systems, and event-driven gameplay mechanics.

## Features

* First Person Controller
* Weapon Switching System
* Pistol, Machine Gun, and Sniper Rifle
* Weapon Pickups
* Ammo System
* Shield System
* Enemy AI using NavMesh
* Turrets and Projectile Combat
* Explosion VFX
* Sniper Zoom
* Recoil System
* Spawn Gates
* Win/Lose Conditions
* Event Manager Architecture
* Cinemachine Camera Systems

## Technologies Used

* Unity 6
* C#
* NavMesh
* Scriptable Objects
* Cinemachine
* TextMeshPro
* Unity Input System
